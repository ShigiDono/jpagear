JPAGear
=======
## Building
### Requisites

Tested with
*   jsoncpp <b>rev 255</b>
*   libspawner <b>latest rev</b>
<pre><code>
	svn checkout http://spawner.googlecode.com/svn/trunk/libspawner libspawner  
	svn checkout https://jsoncpp.svn.sourceforge.net/svnroot/jsoncpp/trunk/jsoncpp runner/jsoncpp/jsoncpp
</code></pre>

### Build instructions

	mkdir build
	cd build
	cmkake ..
